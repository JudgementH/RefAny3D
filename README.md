## Quickstart

### Create environment
1. Clone the repository and create conda environment: 
```
git clone https://github.com/JudgementH/RefAny3D.git
conda create -n r3d python=3.10
conda activate r3d
```
2. Requirements installation
```
pip install -r requirements.txt
```
